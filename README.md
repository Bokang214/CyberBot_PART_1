# CyberBot_PART_1
Cybersecurity Awareness Chatbot

Project Description
This project is a Cybersecurity Awareness Chatbot developed in C#. It educates users about
phishing, password security, and safe browsing using a console-based menu system.

Features Implemented
- Voice Greeting (WAV)
- ASCII Art Header
- User Personalisation
- Menu System
- Input Validation
- Typing Effect
- Colour Formatting

- Voice Greeting
Uses System.Media.SoundPlayer to play a WAV file at startup.

Code Structure
- Program Class (entry point)
- CyberBot Class (logic and interaction)
  
Topics Covered
- Phishing
- Password safety
- Safe browsing
  
Challenges & Solutions
Challenge: Audio playback.
Solution: Installed required package and used WAV format.

Conclusion
This chatbot demonstrates how simple applications can promote cybersecurity awareness.
